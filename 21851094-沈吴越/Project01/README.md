# 编程作业1：OpenGL基本程序

## 作业要求：
- 使用glfw和glew搭建OpenGL基本框架（也可以使用glut或者Qt等其他框架）
- 实现基础绘制（本文绘制了茶壶）
- 实现相机控制（本文实现了模型的不同视角、不同观察距离及模型平移功能）

### 作业成果：
#### 功能:
- 茶壶基础模型的绘制；
- 鼠标左键拖动可以实现模型的旋转；
- 鼠标中键拖动可以实现不同的观察距离；
- 鼠标右键拖动可以实现模型的平移；
- 作业最终的效果演示见Demo
