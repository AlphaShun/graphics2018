# Project 3

基于原生GLUT框架加载的鼠标键盘模型（obj格式），并添加了两个光源

支持通过鼠标移动进行平移旋转和缩放

支持键盘事件进行场景漫游：q、e、a、d（调整观察点位置）、w、s（调整视点高度）

## 环境
macOS Mojave 10.14.1、Xcode 10.1、macOS原生OpenGL.framework、macOS原生GLUT.framework
