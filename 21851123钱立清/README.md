## 开发环境

* 系统 Mac OS X 10.14
* IDE Xcode 9.2

## 编程作业1：OpenGL基本程序

* 使用glfw和glew搭建OpenGL基本框架
  * 也可以使用glut或者Qt等其他框架
* 实现基本模型绘制
  * 绘制立方体、圆球或者茶壶
* 实现相机控制
  * 支持从不同视角和具体观察模型

### 作业成果

使用Glut框架实现一个简单的茶壶模型，支持鼠标、键盘进行旋转、缩放等功能。

## 编程作业2：OpenGL绘制星球

* 必要特性
  - 1、光照
      * 太阳为光源
  - 2、纹理
      * 使用图片进行纹理映射
* 可选特性
  - 3、使用顶点着色器和片段着色器
      * 自己实现光照效果
  - 4、鼠标选择
      * 点击不同球体显示不同名称

### 作业成果

绘制星球，包括太阳，地球和月球，实现各自的公转、自转和纹理映射，效果如下：
![](https://i.ibb.co/NNR64d0/1546337072489.jpg)

## 编程作业3：负责三维模型显示

* 1、支持以下三维模型文件格式中至少1种：
  * OBJ
  * 3DS
  * STL
* 2、支持多个光源的光照效果
* 3、支持多种视点游览方式
  * 以模型为中心的平移旋转和缩放
  * 以视点为中心的场景漫游
