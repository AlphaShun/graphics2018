# 编程作业1：OpenGL基本程序

本次作业包括三部分：
- 使用glfw和glew（也可以使用glut或者Qt等其他框架）搭建OpenGL基本框架
- 实现基本模型绘制
- 实现相机控制

我的作业环境：
- MacOS Mojave
- OpenGL 3.3
- GLUT

## IDE及OpenGL等框架配置

IDE使用的XCode，配置参考
- [Mac 下XCode 配置GTools／GLFW ／GLEW 做OpenGL 开发工作](https://www.jianshu.com/p/25d5fbf792a2)

- [LearnOpenGL-CN](https://learnopengl-cn.github.io/)
