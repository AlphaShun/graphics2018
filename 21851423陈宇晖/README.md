# 编程作业1：OpenGL基本程序

本作业使用glfw和glew搭建OpenGL基本框架

IDE使用的是Visual Studio 2017

实现了以下功能：

- create a window
- use shader
- add camera
- accomplish keyboard callback
- accomplish mouse callback

- add texture

- add phong light model
- add a skybox
- add Specular Map
- add light source of 3 different type
