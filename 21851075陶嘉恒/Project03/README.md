# 编程作业3：复杂三维模型显示

参考:
- [1](https://blog.csdn.net/arag2009/article/details/78099974) 模型加载

实现内容：

OPENGL三维场景搭建、漫游、交互。包括凹凸地形，键盘交互、鼠标交互、树木，火焰等。

1、导入多个OBJ格式三维模型文件格式

2、支持多个光源的光照效果

3、支持多种视点浏览方式
- 以模型为中心的平移旋转和缩放
- 以视点为中心的场景漫游
